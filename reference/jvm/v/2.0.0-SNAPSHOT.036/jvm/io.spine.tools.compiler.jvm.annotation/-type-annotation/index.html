protected val subject: ClassName?

The class or enum class name to be annotated. Cannot be used together with file. If both subject and file are specified, IllegalArgumentException will be thrown. If both subject and file are null, the annotation is applied to all the classes passed to this annotation. If not null, the annotation is applied only to the types specified by this field.
