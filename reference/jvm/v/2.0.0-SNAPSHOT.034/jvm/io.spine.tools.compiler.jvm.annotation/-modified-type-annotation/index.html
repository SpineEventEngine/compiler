ModifiedTypeAnnotation

open class ModifiedTypeAnnotation(val generator: String = CLI_APP_CLASS, val addTimestamp: Boolean = false, val commenter: (SourceFile<Java>) -> String? = null) : TypeAnnotation<Modified> (source)

Adds the io.spine.annotation.Modified annotation to the top-level declaration of each Java file in the source set.

The annotation is needed to inform the developers on the fact that the original code generated by Protobuf Compiler and annotated with com.google.protobuf.Generated annotation was also modified by the Spine Compiler and its plugins.

Deriving classes are likely to inherit the class to pass values to the properties, providing a no-argument constructor, as required for a Renderer.
