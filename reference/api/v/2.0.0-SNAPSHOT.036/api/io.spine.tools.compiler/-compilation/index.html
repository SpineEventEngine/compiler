Compilation

object Compilation(source)

Provides functions to report compilation errors and warnings.

The object has two modes of handling the compilation error. In the production mode, the error method prints an error message to System.err and exits the process with ERROR_EXIT_CODE.

In the testing mode, the error method throws Compilation.Error exception with the same error message as printed to the console in the production mode.

The execution mode is detected via the Tests environment type.
