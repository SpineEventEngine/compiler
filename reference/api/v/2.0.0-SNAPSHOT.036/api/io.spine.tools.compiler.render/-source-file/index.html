SourceFile

class SourceFile<L : Language>(source)

A file with the source code.

This file is a part of a source set. It should be treated as a part of a software module rather than a file system object. One SourceFile may reflect multiple actual files on a file system. For example, a SourceFile may be read from one location on the file system and written into another location.

Parameters

the language of this source code file.
