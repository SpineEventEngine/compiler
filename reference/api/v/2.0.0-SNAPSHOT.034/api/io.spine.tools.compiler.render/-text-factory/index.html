TextFactory

object TextFactory(source)

Static factories and precondition checks for creating instances of Text.

A recommended way for using this class is using its methods statically imported, so that the creation of Text objects looks compact:

import io.spine.tools.compiler.render.TextFactory.text
...
var twoLines = text("one", "two");
