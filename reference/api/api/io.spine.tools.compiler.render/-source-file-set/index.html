SourceFileSet

class SourceFileSet : Iterable<SourceFile<*>> (source)

A mutable set of source files that participate in the code generation workflow.

The initial set of files is obtained when the source set is loaded the inputRoot directory.

The code generation process may add new files to the set, or delete existing ones.

The resulting files are written to the outputRoot directory.

When files are deleted, they still remain in the set, but are marked as deleted. Actual deletion happens when the set is written.

The source set can be configured to perform some actions before reading the files.
