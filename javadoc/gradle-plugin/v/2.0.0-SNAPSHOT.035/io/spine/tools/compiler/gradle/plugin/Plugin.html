    The Gradle plugin of the Spine Compiler.

    Adds the launchSpineCompiler tasks which runs the executable with the arguments assembled from settings of this plugin.

    The users can submit configuration parameters, such as renderer and plugin class names, etc. via the compiler { } extension.

    The user classpath to the Compiler can be passed by declaring dependencies using the spineCompiler configuration.

    Example (build.gradle.kts):

    spine {
    compiler {
        plugins("com.acme.MyPlugin")
    }
}

dependencies {
    spineCompiler(project(":my-plugin"))
}
