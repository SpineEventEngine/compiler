public abstract class LaunchSpineCompiler
extends JavaExec

    A task which executes a single Spine Compiler command.

    This class is public to allow users to find the Compiler tasks by their type. This is useful to configure task dependencies, enable and disable individual tasks, add conditions via onlyIf { }, etc.

    Users should NOT change the CLI command, user directory, etc. directly. Please refer to the compiler { } extension configuring the Compiler.
