      • byCode

        @VisibleForTesting() final <L extends Language> SourceFile<L> byCode(Path relativePath, String code)

        Constructs a file from source code and casts the instance assuming the code belongs to the language specified by the generic parameter L.

        The primary purpose of this function is to avoid casts at the call sites when the language is surely known. In other cases, please use fromCode.

        Parameters:
        relativePath - the file system path for the file relative to the source root; the file might not exist on the file system.
        code - the source code.
