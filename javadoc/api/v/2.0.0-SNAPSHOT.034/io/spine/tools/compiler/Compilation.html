Package io.spine.tools.compiler

Object Compilation

  • All Implemented Interfaces:
    
public class Compilation

    Provides functions to report compilation errors and warnings.

    The object has two modes of handling the compilation error. In the production mode, the error method prints an error message to System.err and exits the process with ERROR_EXIT_CODE.

    In the testing mode, the error method throws Compilation.Error exception with the same error message as printed to the console in the production mode.

    The execution mode is detected via the Tests environment type.

      • error

         final Void error(File file, Integer line, Integer column, String message)

        Prints the error diagnostics to System.err and terminates the compilation.

        The error message includes the path to the file, line and column numbers followed by the message.

        The termination of the compilation in the production mode is done by exiting the process with ERROR_EXIT_CODE.

        If the code is run under tests the method throws Compilation.Error.

        Parameters:
        file - The file in which the error occurred.
        line - The one-based number of the line with the error.
        column - The one-based number of the column with the error.
        message - The description of what went wrong.
